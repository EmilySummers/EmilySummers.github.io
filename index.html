Solo Project ◉ 1 week

Roam is a trip planning app with a front-end built using React and a back-end using Django and Python. Users can create boards to organise upcoming trips or collate memories of past trips by pinning photos, posting links to attractions and adding to the to-do list. There are also map and current weather features that I built using the Mapbox and OpenWeather APIs.

Users have the ability to share boards with others attending the trip or make the trip open to all so that other users can request to join. Each user has a notifications page where they can view and manage their pending offers and invitations. The project was styled using Sass and Canva.

Group Project ◉ 1 week

Chow Vow is a MERN stack app aimed at those wanting to connect with others based on a shared love of cooking and to swap culinary skills. Users are able to search for other chefs based on their location or a specific skill they wish to learn, then once connected, they can share their cooking skills with each other.

I took the lead on developing the index page that filtered users depending on the skill selected and creating the star ratings feature where users can rate each other. I built the back-end functionality for the user profile and ratings using Node, including a function that collected all of the users ratings and calculated the average, storing the result in a virtual schema. Using React, I built the front-end for the homepage, navbar and index page and styled them, together with the offers page, using Sass.
